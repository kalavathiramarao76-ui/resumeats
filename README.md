# ResumeATS

Free ATS Resume Builder & Scanner. Build resumes that pass Applicant Tracking Systems.

## Features
- Guided resume builder with live preview
- Instant ATS compatibility score (20+ checks)
- Keyword matching against job descriptions
- PDF export
- Auto-save (localStorage)
- 100% client-side — your data never leaves your browser

## Tech Stack
Next.js 14, TypeScript, Tailwind CSS, jsPDF, html2canvas
